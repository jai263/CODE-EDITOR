HTML, CSS & JavaScript Basics

HTML

HTML (HyperText Markup Language) forms the structure of web content. It uses tags to define elements such as headings, paragraphs, links, and images. Here are some common HTML tags:

  • <h1> to <h6>: Heading tags, with <h1> being the largest and <h6> being the smallest.
  • <p>: Paragraph tag for blocks of text.
  • <a>: Anchor tag for hyperlinks.
  • <img>: Image tag for embedding images.
  • <br>: Line break tag.
  • <hr>: Horizontal rule tag.
  • <div>: Division tag for grouping elements.

Example:

<h1>This is a heading</h1>
<p>This is a paragraph.</p>
<a href="https://example.com">This is a link</a>
<img src="image.jpg" alt="Description">
<br>
<hr>
<div>This is a division</div>

CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls the appearance of HTML elements, such as colors, fonts, and spacing. Here are some common CSS properties:

  • color: Sets the color of text.
  • background-color: Sets the background color of an element.
  • font-family: Sets the font of text.
  • font-size: Sets the size of text.
  • margin: Sets the space outside an element.
  • padding: Sets the space inside an element.
  • border: Sets the border of an element.
  • width and height: Sets the width and height of an element.
  • display: Controls how an element is displayed (e.g., block, inline, none).

Example:

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
}

h1 {
    color: #333;
}

JavaScript

JavaScript is a programming language that adds interactivity to web pages. It allows you to create dynamic content, such as sliders, forms, and animations. Here are some common JavaScript methods and properties:

  • document.getElementById(): Selects an element by its ID.
  • document.querySelector(): Selects the first element that matches a CSS selector.
  • document.querySelectorAll(): Selects all elements that match a CSS selector.
  • element.innerHTML: Gets or sets the HTML content of an element.
  • element.style: Gets or sets the inline CSS styles of an element.
  • element.addEventListener(): Attaches an event handler to an element.
  • alert(): Displays an alert box with a message.
  • console.log(): Outputs a message to the browser console.

Example:

document.getElementById('myButton').addEventListener('click', function() {
    alert('Button clicked!');
});
